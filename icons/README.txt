Tab Wrapper Extension Icons
============================

This folder should contain three icon files for the Chrome extension:

Required Icons:
--------------
1. 16.png  (16x16 pixels) - Used in the toolbar and favicon
2. 48.png  (48x48 pixels) - Used in the extensions management page
3. 128.png (128x128 pixels) - Used in the Chrome Web Store and installation

Design Guidelines:
-----------------
- Style: Modern, clean, minimalist
- Colors: Purple gradient (#7c3aed to #6d28d9) as primary, with white accents
- Concept: Should represent tab organization/grouping
- Background: Transparent PNG format

Icon Concept Ideas:
------------------
1. Stack of papers/files being organized into groups
2. Browser tabs with grouping brackets
3. Folder icon with tab symbols
4. Grid layout representing organized tabs
5. AI brain with tab symbols

How to Create the Icons:
-----------------------
1. Use a design tool like Figma, Adobe Illustrator, or Canva
2. Create a square canvas for each size (16x16, 48x48, 128x128)
3. Design with the purple gradient theme
4. Export as PNG with transparent background
5. Test the icons at different sizes to ensure clarity

Alternative: Use an AI Image Generator
-------------------------------------
You can use tools like Midjourney, DALL-E, or Stable Diffusion with prompts like:

"Chrome browser extension icon, minimalist design, purple gradient, tab organization concept, 128x128 pixels, transparent background, professional UI design"

File Naming:
-----------
Save the files exactly as:
- 16.png
- 48.png  
- 128.png

Testing:
--------
After creating the icons, load the extension in Chrome to verify they appear correctly:
1. Open chrome://extensions/
2. Enable Developer mode
3. Click "Load unpacked" and select the extension folder
4. Check the icon appears in the toolbar and extensions page

Note: The icons are referenced in manifest.json. If you use different filenames, update the manifest accordingly.

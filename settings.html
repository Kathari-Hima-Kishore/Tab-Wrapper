How to get your Gemini API key:
  1. Visit Google AI Studio

  3. Click "Create API Key" or select an existing one
  4. Copy the API key (it starts with "AIza...")
  5. Paste it in the field above and click "Save"
  6. The API key is stored locally and never sent to anyone except Google's servers
About Tab Wrapper

Tab Wrapper uses Google's Gemini 2.5 Flash AI to intelligently organize your browser tabs into logical groups. The extension analyzes each tab's content and automatically creates native Chrome tab groups with descriptive names and colors.

Features:

  • AI-powered tab analysis and grouping
  • Creates 2-6 specific, descriptive groups
  • Uses native Chrome tab groups
  • Works with any web pages
  • Privacy-focused (API key stored locally)

Privacy: Your tab data is sent to Google's Gemini API for analysis but is not stored anywhere. The API key is stored only in your browser's local storage.
